--------製品のインストールやその後の設定について--------

・Python本体をインストールします。
※(Microsoft-Store版は環境変数(＝プログラムの実行パス)を設定してくれないので、公式サイトからダウンロードしてインストールするのをお勧めします。


・既成のモジュール＆ライブラリィをインストールします。ターミナル上で以下のコマンドを実行してください。
※その他のモジュール＆ライブラリィはPython本体に同梱しているので別途インストールする必要はありません。
※「requirments.txt」ファイルに以下のモジュール＆ライブラリィの名前を書いた上でpipインストールすれば一度にまとめてインストールできます。
「py -m pip install pytz」
(もしくは「pip3 install pytz」)

「py -m pip install Flask」
(もしくは「pip3 install Flask」)

「py -m pip install Flask-Paginate」。
(もしくは「pip3 install Flask-Paginate」)

「py -m pip install pysqlite3」。
(もしくは「pip3 install pysqlite3」)


・Windowsのスタートアップにプログラムを登録しておきます。
※「スタートアップ」フォルダーにプログラム本体のショートカットを入れるだけです。


・webブラウザー上で「(http://)localhost:5000」でプログラムにアクセスします。
※簡単にアクセスするためにブックマークしておくことをお勧めします。
※あるいは、デスクトップ上にURLショートカットを作成しておくと良いです。


・管理者用パスワードを変更したければ、プログラムファイル「main.py」の先頭部分に書かれている以下の部分を書き換えてください。
※「ADMIN_PASS_WORD = 'gp1192'」(ディフォルト値)。


・管理者用パスワード以下のコメント付きの各行の値を変更することで、プログラムの動作や仕様を変更できます。
※「SESSION_TIME = 20」(セッションの持続時間)、
「USERS_DATABASE = 'app_usrm.db'」(ユーザー情報のデータベースファイル)、
「ATTENDANCE_DATABASE = 'app_tmm.db'」(勤怠・打刻情報のデータベースファイル)、
「ITEM_PER_PAGE = 40」(一画面ごとに表示する項目の数)、
「IMPORT_PATH = './cache_data'」(インポートするCSVファイルの場所)、
「EXPORT_PATH = 'cache_data/export_attendance.csv'」(エクスポートするCSVファイルの場所)。


・「FLASK-ENV」環境変数の「environment: development」から「environment: product」に変更することで、プログラムの実行速度が向上します。


・誤ってデータベースファイルを削除してしまったり、データベースファイルが破損してしまった場合でもプログラムによって実行時に自動で再作成されます。
※「DBファイルにアクセスできません, プログラムファイルを新規のプロジェクトフォルダに移してください.」というエラーメッセージがプログラムで表示されたら、
新規のプロジェクトフォルダーを作成・用意して、そこに元のプログラムファイル一式をコピーないし移動させてください。それでエラーが解消されるはずです。
